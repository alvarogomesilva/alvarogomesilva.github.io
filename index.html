O que estudo?

  • HTML

    HTML é a linguagem que estrutura páginas da web, ideal para criar portfólios organizando textos e imagens de forma clara.

  • CSS

    CSS é a linguagem usada para estilizar páginas da web. Em um portfólio, o CSS define cores, fontes e layouts, tornando o design visualmente atraente e personalizado.

  • JavaScript

    JavaScript é uma linguagem que adiciona interatividade às páginas web, tornando portfólios mais dinâmicos e envolventes.

  • React

    React é uma biblioteca JavaScript para construir interfaces de usuário. Em portfólios, permite criar componentes reutilizáveis, tornando o desenvolvimento mais eficiente e a experiência do usuário mais fluida.
